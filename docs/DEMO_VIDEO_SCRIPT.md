# LevelLens Demo Video Script

Target length: 2 minutes 30 seconds. Narration and on-screen labels should be English.

## 0:00-0:20 - Problem and product

"Teachers often need one lesson in several reading levels, but a normal AI rewrite does not prove that the result actually fits the selected level. LevelLens turns one teaching text into verified reading-level versions for English, Spanish, and Japanese."

Show the New material screen and the language/level controls.

## 0:20-0:55 - Create a result

"I can paste a teaching text or import one public article URL. I select two target levels and start the conversion. Each level runs independently, so a slow or failed level does not block the others."

Paste an original sample, select two levels, and start a prepared job or show a completed job. Do not wait silently for a long live generation.

## 0:55-1:35 - Explain the differentiator

"The important difference is verification. GPT-5.6 writes each level-specific draft, but LevelLens scores it deterministically on the server. If a draft misses the target, the numeric score becomes revision feedback for GPT-5.6. The result shows the final score, target range, and number of attempts."

Show the original/simplified comparison and Readability card. Point to a result with at least two attempts when available.

## 1:35-2:00 - Teacher workflow

"LevelLens also checks factual consistency, highlights teachable language patterns instead of only topic words, and creates four-choice comprehension questions. Teachers can inspect the questions and export a student or teacher PDF worksheet."

Show Fact consistency, Language focus, Questions, then Export/PDF preview.

## 2:00-2:25 - Codex and GPT-5.6

"I built LevelLens with Codex and GPT-5.6. Codex accelerated the Prisma pipeline, deterministic scorer tests, multilingual UI, PDF export, safe URL import, and public-hosting safeguards. GPT-5.6 is used through Structured Outputs for rewriting, fact checking, language focus, and questions. The README includes the dated commits and Codex session evidence."

Show the README section and optionally a short git log view.

## 2:25-2:30 - Closing

"LevelLens makes differentiated reading material faster to prepare and easier to verify."

Show the completed Result screen and product name.
