# LevelLens

[English](README.md) | [Español](README.es.md) | 日本語

LevelLensは、英語・スペイン語・日本語の教材を複数の読解レベルへ変換する教師向けツールです。GPT-5.6による生成だけでなく、決定論的な可読性スコア、再生成による目標範囲の検証、事実整合性確認、文法・文型中心の言語のポイント、4択読解問題を提供します。

## 主な機能

- テキスト貼り付け、または公開HTML記事URLの取り込み。
- 1〜4件の対象レベルを並列生成。
- 原文と変換文の左右比較。
- EN・ES・JAそれぞれに対応した決定論的な可読性検証。
- 事実整合性、言語のポイント、4択読解問題。
- 生徒用・教師用PDFワークシート出力。
- 英語・スペイン語・日本語のUI切替。
- 14日間の匿名ワークスペース保護と自動データ削除。

## CodexとGPT-5.6

Codexは、アプリ基盤、決定論的スコアラー、生成パイプライン、UI、PDF出力、公開ホスティング向け保護の実装に利用しました。GPT-5.6は、Structured Outputsを用いてサーバー側で変換、事実確認、言語のポイント、設問生成に利用しています。主要なCodexセッションIDは`019f6fca-6bee-7f33-9836-ae5f6f5fa65b`です。
